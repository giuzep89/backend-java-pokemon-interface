# Pokemon Battle Simulator

A small vanilla Java app built on top of a base written by my teachers
at Novi Hogeschool to learn to use inheritance and interfaces. You pick
your Pokemon, enter a gym, and battle the gym owner in turn-based combat.

## What's in here?

- 4 Pokemon types (Fire, Water, Grass, Electric) with type advantages
- Turn-based battles against a gym owner
- Special attacks like leechSeed (steals HP) and rainDance (heals grass types)
- Feed your Pokemon mid-battle for an HP boost

## Tech stack

- Java (no frameworks, just plain Java)
- Command-line interface

## How to run

```bash
cd src
javac *.java
java Main
```
